## Dialex 是什么

**核心比喻：** 想象一张圆桌，不同书籍的作者们围坐讨论——而讨论本身产生了一本全新的书，这本书是任何一位作者都无法独自写出的。

Dialex 是一个多智能体系统：多个 AI 专家心智——每个都拥有独特的推理方法——围绕问题展开辩论，最终产生任何单一心智都无法独立生成的知识。

### 诚实定位

Dialex 是一个**认知搜索放大器**。它使用多个分析视角来更广泛地探索问题空间，通过结构化综合来组合结果，并呈现突出分歧、张力和新颖组合的发现。

| 它是什么 | 它不是什么 |
|---------|----------|
| 约束条件下的功能性多样化 | 心智之间的"真正分歧" |
| 组合性和结构性新颖 | 本体论层面的涌现 |
| 结构化分歧与综合 | 具有真实身份的存在性辩论 |
| 人格保真度 | 认知克隆 |

> 系统必须通过零假设测试：如果它产生的结果不明显优于单个 LLM 被提示"从多个角度分析 X，考虑张力和矛盾"的结果，那么这个架构就不值得其复杂性。

## 核心特性

- **多智能体辩论引擎** — 具有不同推理方法的心智围绕问题进行结构化辩论
- **自组织知识图谱** — SQLite + JSON 存储，随每次辩论而增长；知识不断积累和关联
- **三级涌现检测** — 生产性综合（L1）、概念新颖性（L2）、结构性涌现（L3）
- **Belnap 四值逻辑** — 矛盾不是错误，而是最有价值的知识来源
- **双时态模型** — 每个知识节点都有时间有效性和历史上下文
- **基于张力的检索** — 搜索的不仅是相似内容，还有具有生产性差异的内容
- **本地优先** — 无需外部基础设施依赖；SQLite + JSON + git

├─────────────────────────────────────────────┤
│  第4层：身份配置文件                           │
│  每个心智的认知配置文件                         │
│  推理方法 · 语音 DNA                          │
├─────────────────────────────────────────────┤
│  第3层：语义检索                              │
│  基于张力的搜索 · 跨心智检索                    │
│  矛盾检索 · 聚类                              │
├─────────────────────────────────────────────┤
│  第2层：知识图谱                              │
│  SQLite + JSON · Belnap 四值逻辑              │
│  双时态 · 溯源                                │
├─────────────────────────────────────────────┤

### 图谱节点类型

| 类型 | 来源（DNA Mental 层） |
|-----|---------------------|
| `pattern` | L1-L4：行为模式、沟通风格、习惯、识别模式 |
| `mental_model` | L5：心智模型与框架 |
| `heuristic` | L4-L5：决策规则与认知捷径 |
| `value` / `belief` | L6-L7：价值观层级与核心执念 |
| `contradiction` | L8：生产性悖论（"黄金层"） |
| `synthesis` | 由辩论生成——涌现知识 |

## 科学原理

Dialex 基于 7 项研究原则：

1. **认知多样性** — 解决方案的质量取决于推理方法的多样性，而非智能体的数量。3个采用不同方法的心智优于10个采用相同方法的心智。

2. **生产性张力** — 不同视角之间的矛盾是有价值的信号，不是需要消除的错误。当两个心智产生分歧时，正是最有趣的知识隐藏之处。

3. **Belnap 四值逻辑** — `true`、`false`、`both`、`neither`——允许表示矛盾知识而不坍缩为单一真相。一个概念可以在不同参考框架下同时为真和为假。

4. **双时态模型** — 所有知识都有两个时间轴：在真实世界中何时有效，以及系统何时记录。支持"在Y日期我们对X的认知是什么"这类查询。

5. **弱涌现（Bedau）** — 涌现的内容原则上可推导但实践中无法预测；新颖性来自重组，而非魔法。系统不是发明——而是发现隐含存在的组合。

6. **时间衰减** — 知识具有半衰期；未被使用的知识会逐渐降低置信度。参与辩论会更新节点的相关性。

7. **反谄媚机制** — 主动防止心智人为地趋向共识。轻易达成的一致不会产生新知识。

## 构建基础：MMOS

Dialex 构建于 **MMOS（Mind Mapping & Orchestration System）** 之上，由 **Alan Nicolas** 创建。

MMOS 是上游系统，通过 **DNA Mental** 8层方法论创建专家心智——这是一个严格的流程，映射领域专家的行为模式、沟通风格、心智模型、价值观、执念和生产性悖论。

Dialex 接收 MMOS 创建的这些心智，赋予它们此前不具备的能力：**相互辩论**，并产生超越任何单一心智的知识。

> **致谢：** Alan Nicolas 是 MMOS 和 DNA Mental 方法论的创建者。Dialex 将他的工作扩展到涌现集体智能领域。

## 研究与致谢

本项目基于广泛的研究：

- **50+ 篇学术论文** 在初始研究阶段由 5 个专业智能体团队完成分析
- **35 个候选框架** 由 4 个分析子智能体评估（KARMA、TruthfulRAG、MAGMA、Graphiti、Kumiho、GKMAD、DoM/DoG、MAKGED、HippoRAG、Agentic-KGR 等）。其中 5 个被纳入 MVP，10 个延迟至 v2，21 个被淘汰
- **5 轮对抗性评审** 来自不同视角（务实架构师、认知科学家、产品创业者、红队测试者、心灵哲学家）
- **4 项失败分析** 识别并缓解了 SIF（语义隔离失败）模式

主要理论来源：

| 作者/理论 | 对 Dialex 的贡献 |
|----------|----------------|
| Belnap（四值逻辑） | 知识库中不一致性的处理 |
| Dung（论辩框架） | 智能体间辩论的形式化结构 |
| Bedau（弱涌现） | 定义和检测新颖性的理论框架 |
| Bakhtin（杂语理论） | 多元声音作为意义来源 |
| Schwartz（价值观分类） | 图谱价值观节点的分类体系 |
| Dreyfus（专业技能模型） | 启发式规则的置信度级别 |
